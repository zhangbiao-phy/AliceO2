Detailed Description

template<typename DataT = double, unsigned int Nx = 129, unsigned int Ny = 129, unsigned int Nz = 180>
class o2::tpc::RegularGrid3D< DataT, Nx, Ny, Nz >

Template Parameters
DataTthe type of data which is used during the calculations
Nxnumber of vertices in x direction
Nynumber of vertices in y direction
Nznumber of vertices in z direction

This class implements basic properties of a regular 3D-Grid like the spacing for each dimension and min and max coordinates.
