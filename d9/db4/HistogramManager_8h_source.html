51  // Create a new histogram class

53  // Create a new histogram in the class <histClass> with name <name> and title <title>
54  // The type of histogram is deduced from the parameters specified by the user
55  // The binning for at least one dimension needs to be specified, namely: nXbins, xmin, xmax, varX which will result in a TH1F histogram
56  // If the value for a variable is left as -1 then that is considered to not be used
57  // Up to 3 dimensional histograms can be defined with this function
58  // If isProfile = true, the last specified variable is the one being averaged
59  // For the case of TProfile3D, the user must use the varT to specify the averaged variable
60  // If specified, varW will be used as weight in the TH1::Fill() functions.
61  // If specified, the xLabels, yLabels, zLabels will be used to set the labels of the x,y,z axes, respectively.
62  // The axis titles will be set by default, if those were specified (e.g. taken from a Variable Manager)
63  // Otherwise these can be specified in the title string by separating them with semi-colons ";"

68  const char* xLabels = "", const char* yLabels = "", const char* zLabels = "",
69  int varT = -1, int varW = -1);
